간단하고 안전한 마우스 제스처 확장프로그램

마우스 우클릭 + 드래그만으로 빠르게 브라우저를 조작하세요.

[ 지원 제스처 ]
← 왼쪽으로 드래그: 뒤로가기
→ 오른쪽으로 드래그: 앞으로가기
↓→ 아래로 + 오른쪽: 탭 닫기

[ 특징 ]
- 개인정보 수집 없음: 외부 서버 통신이 전혀 없습니다
- 오픈소스: 모든 코드가 공개되어 있어 누구나 검증 가능
- 가볍고 빠름: 최소한의 권한만 사용 (tabs 권한 1개)
- 시각적 피드백: 제스처 경로가 화면에 표시됩니다

CRXmouse, Gesturefy 등의 개인정보 이슈가 걱정되셨다면, 직접 코드를 확인할 수 있는 Simple Mouse Gestures를 사용해보세요.
